# Design System

Direction: **quiet wall art, grid-first, dark-first, minimal, reflective, no dashboard**.

## Core Principles
- The calendar canvas is the product surface.
- Remove SaaS chrome: no sidebar, no heavy nav, no dashboard cards.
- Prioritize calm spacing, restrained contrast, and legible rhythm.

## Typography
- Primary font: `Outfit`.
- Simple hierarchy: subtle scale, medium weight emphasis.

## Color and Mood
- Dark-first palette with neutral-cool tones.
- Soft borders and low-glow accents.
- Avoid loud saturation and ornamental gradients.

## Layout
- Grid-oriented composition.
- Centered canvas with generous margins.
- Minimal top framing only when required.
