# Roadmap

## Phase 1 - Foundation (Done)
- Repo structure
- Web/API/shared scaffolding
- Docs and conventions
- Minimal app shell

## Phase 2 - Core Data + Storage (Done)
- Prisma + SQLite setup
- Models: `Profile`, `LifeEvent`, `AppSettings`

- Fullscreen calm grid shell
- Week-cell rendering from engine output
- Minimal interaction scaffolding

## Phase 5 - Product Extensions
- Export workflows
- Personalization refinements
